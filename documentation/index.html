Vue Modern Landing Page

A modern, clean, and fully responsive Vue 3 landing page template built with Vite, Tailwind CSS, and Pinia.

Introduction

This template is designed for startups, SaaS products, portfolios, and marketing websites. It is frontend-only and focuses on clean code, performance, and easy customization.

Requirements

  • Node.js 18+
  • npm or yarn

Installation

npm install
    npm run dev

To build for production:

npm run build

Project Structure

src/
    ├─ components/   UI & layout components
    ├─ sections/     Page sections (Hero, Features, Pricing)
    ├─ layouts/      Layout wrappers
    ├─ views/        Pages (Home, Contact, Privacy, Terms)
    ├─ stores/       Pinia stores
    ├─ data/         Editable content
    ├─ router/       Vue Router configuration

Customization

Text Content

All main text content is stored inside: 

src/data/

Example: 

// src/data/contact.js
    export const contactContent = {
    title: 'Get in Touch',
    subtitle: 'We would love to hear from you!',
    }

Colors & Styling

This template uses Tailwind CSS. You can customize colors and styles by editing: 

tailwind.config.js

Contact Page

The contact page is located at: 

src/views/ContactView.vue

Contact page content can be edited here: 

src/data/contact.js
Note: This template is frontend-only. The contact form does not submit data to a server. You can connect it to your own backend or services like Netlify Forms.

Dark Mode

Dark mode is implemented using Tailwind's dark: classes and managed via Pinia. The user's preference is stored locally and respects system settings.

Deployment

You can deploy this template easily using platforms like Netlify or Vercel.
